🔋 Battery-Powered Smart Grid Optimizer with Safe AI

A Reinforcement Learning–based system that intelligently manages home energy usage using a high-fidelity virtual Tesla Powerwall 2 model.

🚀 Overview
Modern households often waste excess solar energy during midday and rely heavily on grid power after sunset. This project addresses that inefficiency by using Safe Reinforcement Learning to optimize energy storage and usage in real time.
The system leverages PPO (Proximal Policy Optimization) combined with CMDP (Constrained Markov Decision Process) to ensure not only optimal performance but also safe and sustainable battery operation.

💡 Key Features

🌞 Learns from real-world solar irradiance and household consumption data (Telangana dataset)

⚡ Dynamically adjusts battery charging and discharging based on predicted solar availability

🧠 Implements Safe RL (PPO-Lagrangian + CMDP) for constraint-aware decision-making

🔋 Uses a physics-accurate digital twin of the Tesla Powerwall 2

♻️ Minimizes unnecessary battery cycling to extend battery lifespan

🏡 Enables efficient, safe, and intelligent home energy management
